# Retail Sales Analysis

تحليل بيانات مبيعات متجر تجزئة باستخدام Python وPandas.

## الهدف
- استخراج رؤى تساعد في اتخاذ قرارات تسويقية
- تحديد المنتجات والمواسم الأفضل مبيعًا
- اقتراح استراتيجيات تسعير وترويج

## المحتويات
- تحليل استكشافي EDA
- تنظيف البيانات ومعالجة القيم المفقودة
- التصور البياني للمبيعات
- التوصيات النهائية

## الأدوات المستخدمة
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
